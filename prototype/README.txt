# Creating the directory
  Create a new directory and unzip the zip file into it

# Installing the requirements
  Install NodeJs and npm:
    - For help, use the following link:
      https://websiteforstudents.com/install-the-latest-node-js-and-nmp-packages-on-ubuntu-16-04-18-04-lts/ 
    - Instaling NodeJs should also install npm

  Install npm requirements:
    - Run the following commands to install the npm requirements:
      npm install react
      npm install react-dom
      npm install react-scripts
      npm install react-bootstrap
      npm install react-router-dom

# Start the server
  Deploy locally by running the following command:
    npm start
